## 🛠️ Personalización

¿Quieres añadir más partidos o cambiar los pronósticos?

Simplemente edita el archivo `index.html`:

1. Busca la sección del deporte que quieres modificar (`#futbol`, `#baloncesto`, `#tenis`)
2. Copia y pega una tarjeta `.partido-card` existente
3. Cambia los nombres, estadísticas y pronósticos

### Ejemplo de tarjeta de partido:
```html
<div class="partido-card">
    <span class="liga-tag">⚽ La Liga • Sáb 19/4</span>
    <div class="equipo equipo-local">
        <div class="equipo-nombre">Real Madrid</div>
        <div class="equipo-stats">Fuerza: 95</div>
    </div>
    <div class="vs">
        <div class="vs-text">VS</div>
        <span class="prediccion gana-local">Gana Local</span>
        <div class="marcador-previsto">2 - 0</div>
    </div>
    <div class="equipo equipo-visitante">
        <div class="equipo-nombre">Betis</div>
        <div class="equipo-stats">Fuerza: 79</div>
    </div>
</div>
```

## 🌐 Despliegue en GitHub Pages

1. **Crea un nuevo repositorio** en GitHub
2. **Sube el archivo** `index.html`
3. Ve a **Settings** → **Pages**
4. Selecciona la rama `main` y carpeta `/root`
5. Guarda y espera unos minutos
6. Tu sitio estará disponible en: `https://TU_USUARIO.github.io/pronosticos-deportivos/`
